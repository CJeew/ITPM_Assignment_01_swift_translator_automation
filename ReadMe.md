## 📌 Project Overview

This project focuses on **testing the accuracy of a Singlish to Sinhala transliteration system**:

🔗 https://www.pixelssuite.com/chat-translator

The goal is to identify **incorrect conversions (FAIL cases)** by testing real-world Singlish inputs and automating the validation process using **Playwright (Python)**.

---

## 🎯 Objectives

- Identify **50 failing test cases**
- Cover **24 Singlish input types**
- Automate test execution using Playwright
- Record results in Excel (Expected vs Actual output)
- Analyze system weaknesses

---

## 🧪 Test Coverage

- ✅ 50 Negative Test Cases (FAIL scenarios)
- ❌ Focus on incorrect transliterations
- 📊 Categories covered:
  - Slang / Casual language
  - Romanization / spelling variations
  - Mixed English + Sinhala
  - Emojis
  - Numbers, dates, time formats
  - Real chat-style sentences
